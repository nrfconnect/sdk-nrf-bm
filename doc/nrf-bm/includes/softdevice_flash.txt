To flash the S115 SoftDevice binary to your device, open the terminal and run the following commands:

1. First, move into the :file:`nrf-bm` working directory:

   .. tabs::

      .. group-tab:: Windows

         .. code-block:: console

            cd C:/ncs/v0.7.0/nrf-bm

      .. group-tab:: Linux

         .. code-block:: console

            cd ~/ncs/v0.7.0/nrf-bm

      .. group-tab:: macOS

         .. code-block:: console

            cd /opt/nordic/ncs/v0.7.0/nrf-bm

#. Then, make sure that your device is erased:

   .. code-block:: console

      nrfutil device erase

#. Finally, flash the SoftDevice binary for the target SoC:

   .. tabs::

      .. group-tab:: nRF54L15

         .. code-block:: console

            nrfutil device program --firmware subsys/softdevice/hex/s115/s115_9.0.0-2.prototype+offset-0x15D000_softdevice.hex

      .. group-tab:: nRF54L10

         .. code-block:: console

            nrfutil device program --firmware subsys/softdevice/hex/s115/s115_9.0.0-2.prototype+offset-0x0DF800_softdevice.hex

      .. group-tab:: nRF54L05

         .. code-block:: console

            nrfutil device program --firmware subsys/softdevice/hex/s115/s115_9.0.0-2.prototype+offset-0x05D000_softdevice.hex
