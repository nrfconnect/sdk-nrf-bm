.. _subsys:

Subsystems
##########

|BMlong| provides subsystems for typical use cases with Nordic Semiconductor devices.

Only the subsystems located under :file:`nrf-bm/subsys` are in the scope of the |BMshort| option.
All other subsystems that are included in the distribution must be ignored when working with the |BMshort| option.
